The Original Classic (1989)

This is the original Prince of Persia game developed by Jordan Mechner for the Apple II between 1985-1989. The source code is available on GitHub, but to run it in a browser, you'll need an Apple II emulator with the game disk image.

Options to play:

  • Use a desktop Apple II emulator (like AppleWin for Windows)
  • Find an online emulator that supports disk image loading
  • Check out the source code repository

Note: The original game requires a disk image file (.dsk) to run in an emulator. The source code repository contains the assembly code, not a ready-to-play disk image.
